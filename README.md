Frontend-NLW3

#  Desenvolvimento
**Pré Requisitos**
- Node.Js 12.20.0


##  Instalação

Primeiro Clone o Projeto
```
git clone git@bitbucket.org:RafaScripts/frontend-nlw3.git
``` 
Entre na pasta do projeto e execute

```
npm install
ou
yarn install ( yarn apenas para quem usa o pacote Yarn )
```

**Executando**
```
npm start
ou
yarn start
```
